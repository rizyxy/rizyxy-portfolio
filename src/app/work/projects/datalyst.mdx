## Overview

Datalyst is a web-based dashboard built for PT Cipta Nirmala with one core purpose: to make data analysis easy. It provides a seamless, user-friendly environment for processing and understanding data, eliminating the need for complex software or specialized skills. With Datalyst, you can visualize trends, discover patterns, and make data-driven decisions with speed and confidence.

## Key Features

- **Data Module**: A module which is responsible for importing, exporting, and processing data such as missing value and outlier handling and much more.
- **Analytic Module**: A module which is responsible for generating insights from the imported data.

## Technologies Used

- **Django**: For creating the web app
