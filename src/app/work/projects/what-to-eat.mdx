## Overview

WhatToEat is a mobile application designed to revolutionize the way people cook at home. By leveraging a comprehensive database of ingredients and recipes, the app empowers users to prevent food waste and save money. The core functionality is simple yet powerful: users input the ingredients they have on hand, and WhatToEat recommends a curated list of delicious and creative recipes they can prepare immediately.

This intuitive approach transforms a common problem—staring at a fridge full of disparate ingredients—into a seamless and rewarding cooking experience. WhatToEat is more than just a recipe app; it's a tool for sustainable living, helping users reduce their environmental footprint while enjoying great food.

## Key Features

- **Dishes Recommendation**: Recommends a list of dishes you can make with your already available cooking ingredients at home
- **Groceries Planner**: Plan on what cooking ingredients you should buy based on your dishes wishlist
- **DIY Recipes**: Create your own recipes and share it with other user

## Technologies Used

This app has 2 version which are the portfolio version and the publicly available version

### Portfolio Version

- **Express JS**: For server app, built using Typescript
- **Flutter**: For mobile app
- **Postgre SQL**: For database

### Publicly Available Version

- **Flutter**: For mobile app
- **Firebase**: For authentication and database using Cloud Firestore
