## Overview

MallHub is a modern mall catalogue app designed to make shopping experiences seamless for both visitors and stores.With MallHub, visitors can quickly browse through categories, discover stores, and find exactly what they need—whether it’s fashion, electronics, or dining —all in one easy-to-use platform.

For stores, MallHub provides a simple way to promote products and attract more visitors without the hassle of traditional marketing. This creates a win–win ecosystem: shoppers save time, while businesses gain visibility.

## Technologies Used

- **Laravel**: For server app and store / admin panel using Filament
- **Flutter**: For mobile app
- **MySQL**: For database
