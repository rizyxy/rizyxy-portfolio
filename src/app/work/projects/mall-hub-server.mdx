## Overview

The MallHub Server app is the powerful, reliable backend infrastructure that drives the entire MallHub ecosystem. It is engineered for high performance and scalability, ensuring a seamless experience for both shoppers and stores, even during peak traffic periods.

## Features

- **Mall Management**:
  Provides a robust, secure interface for mall to easily upload, edit, and manage their store listings.

- **Store Management**:
  Provides a robust, secure interface for mall stores to easily upload, edit, and manage their product listings.

- **REST API**
  Provides a robust, secure REST API for MallHub Mobile App to consume.

## Technologies Used

MallHub Server App is built on top of Laravel, utilizing Filament to build the admin panel.
